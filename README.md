# SimplifyPath Socket Client Testing

## Overview

This repository is dedicated to automated testing for SimplifyPath. The testing process involves establishing a connection to redis and trying to mock ML for predetermined outputs to simplulate end to end application the project also has a Frontend written in pure js launched on localhost:3000

## Getting Started

To set up and run this project, please follow the steps outlined below:

### Prerequisites

Ensure you have the following installed:

- Node.js
